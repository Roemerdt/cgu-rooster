Koppelcode

arrow_back

Het lijkt erop dat u nog nooit eerder deze site heeft gebruikt. Om deze site te laten werken moet u een keer (per jaar) uw koppelcode opgegeven. Wat is een koppelcode? Dit is een code die als beveilig werkt voor uw gegevens, u kunt deze code vinden door naar www.cgu.zportal.nl te gaan en in te loggen, in het onderste icoon van het linkermenu staat de 'koppelapp'. De code heeft deze vorm: 123 456 789 123.
